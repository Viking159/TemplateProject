	Подробная инструкция использования.
Для создания функционала по выбору объектов и выводу их данных необходио:
1) Добавить в Canvas префаб PickInfoPrefab (Prefab/PickInfoPrefab)
2) Настроить префаб:
	- Необходимый размер окна, внешний вид;
	- Свойства сетки  (PickInfoPrefab/Background/Items) объектов (размер, количество строк/столбцов и т.д.);
	- Добавить ячейки объектов (PickInfoPrefab/Background/Items/), дублируя существующую ячейку;
	- Определить размер отображения изображения выбранного элемента (PickInfoPrefab/Background/InfoBackground/ObjImage);
	- Добавить и настроить вывод текста свойств элемента (PickInfoPrefab/Background/InfoBackground/Stats/), дублируя существующие свойства;
	- И прочее.
3) Создать скрипт ScriptableObject для элемента, наследуясь от класса ItemSOBase
4) Создать главный скрипт формы, наследуясь от класса PickerInfoBase
5) Добавить скрипт из п.4 как компонент PickInfoPrefab и установить ссылки:
	- Title - название выбранного элемента (PickInfoPrefab/Background/InfoBackground/TitleText);
	- Obj Image - изображение выбранного элемента (PickInfoPrefab/Background/InfoBackground/ObjImage);
	- Stats Value Text - Массив с текстами значения свойства выбранного элементы (PickInfoPrefab/Background/InfoBackground/Stats/TextStat/statValueText; ...);
	- Items - Элементы (игровые персонажи, предметы для инвентаря, режимы игры, локации и пр.). Необходимо прикреплять ScriptableObject, которые унаследованы от ItemSOBase;
	- Items Image - ячейки отображения элементов(PickInfoPrefab/Background/Items/)
6) Каждой ячеке отображения элемента в компонент Button добавить вызов метода ChangeIndex, передая соответствующий индекс.
7) Сохранить вариант префаба.
8) При необходимом действии создавать объект из префаба, передавая в родительский трансформ - трансформ канваса. Вызвать метод Init у скрипта созданного объекта.

Выбирать элементы можно нажимая на них или с клавиатуры. Для настройки клавиш переключения элементов можно выбрать необходимые клавишы в компоненте Input Manager у формы.
